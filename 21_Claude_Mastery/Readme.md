# 21 — Claude Mastery 🧠

> *From "what is Claude?" to building production multi-agent systems — everything in one place.*

---

## What This Section Is

A complete, dedicated learning path for everything Claude — the model, the tools, the API, and the agent framework. Story-first, beginner to advanced, with code, diagrams, and interview prep at every level.

---

## Four Tracks

| Track | Folder | What You'll Learn |
|-------|--------|------------------|
| 🧠 **Claude as an AI Model** | `01_Claude_as_an_AI_Model/` | How Claude thinks, generates text, was trained, and stays safe |
| 💻 **Claude Code CLI** | `02_Claude_Code_CLI/` | The CLI tool — skills, hooks, memory, MCP, agents, IDE integration |
| 🔌 **Claude API & SDK** | `03_Claude_API_and_SDK/` | Building with the Anthropic SDK — tool use, streaming, caching, cost |
| 🤖 **Claude Agent SDK** | `04_Claude_Agent_SDK/` | Multi-agent systems, subagents, orchestration, safety patterns |
