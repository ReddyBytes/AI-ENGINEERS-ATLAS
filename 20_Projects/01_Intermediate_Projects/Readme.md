## What You'll Have Built by the End

- A semantic search engine over real documents using embedding similarity
- A full RAG pipeline that answers questions from your own files with citations
- A ReAct agent that uses web search, calculator, and Wikipedia tools
- A custom fine-tuned model pushed to HuggingFace Hub with a Gradio demo
- A production RAG system with caching, safety, cost tracking, and RAGAS eval
