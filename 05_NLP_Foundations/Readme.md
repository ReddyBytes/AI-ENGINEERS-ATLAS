# 05 — NLP Foundations

Before any model can understand language, it needs the right tools to handle raw text. This section covers the building blocks every NLP engineer must know.

## What this section covers

| Topic | What you'll learn |
|---|---|
| 01 Text Preprocessing | Clean and prepare raw text for any model |
| 02 Tokenization | Split text into pieces models can process |
| 03 Bag of Words & TF-IDF | Turn text into numbers using word counts |
| 04 Word Embeddings | Give words meaning through dense vectors |
| 05 Semantic Similarity | Measure how close two pieces of text are in meaning |
| 06 Hidden Markov Models | Model sequences with hidden states |
| 07 Conditional Random Fields | Label sequences with context awareness |

## Why start here?

Every modern AI system — from search engines to chatbots to translation apps — starts with these fundamentals. You can't skip them. Even if you jump straight to transformers, you'll keep hitting questions about tokenization, embeddings, and preprocessing.

Start at 01 and work forward. Each topic builds on the last.

## Prerequisites

- Basic Python (lists, loops, functions)
- Section 02 (Math for AI) is helpful but not required
